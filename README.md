# Exploratory Data Analysis (EDA) on Titanic Dataset

## Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data, identify patterns, detect missing values, and visualize important insights.

## Dataset
Titanic Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded the dataset.
2. Displayed the first five rows.
3. Checked dataset information.
4. Generated statistical summary.
5. Identified missing values.
6. Counted survival rate.
7. Created visualizations:
   - Survival Count
   - Passenger Class Distribution
   - Gender Distribution
   - Age Distribution
   - Correlation Heatmap

## Conclusion
The analysis provides insights into passenger survival, age distribution, gender distribution, and passenger classes using statistical summaries and visualizations.
